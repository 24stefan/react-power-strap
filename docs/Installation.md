# Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/24stefan/REACT-POWER-STRAP
````

2. **Navigate to the directory**:

   ```bash
   cd REACT-POWER-STRAP
   ```

3. **Make the script executable**:

   ```bash
   chmod +x react-power-strap.sh
   ```

4. **Run the script**:

   ```bash
   ./react-power-strap.sh
   ```
